 
---------- Download code to System using Git --------

1. git clone https://github.com/BhukyAkash/Automation.git
2. Activate environment in Playwright folder     ----- venv\Scripts\activate
3. Download Playwright extension in VS
4. Install the Playwright through below commands
    python -m pip install pytest
    python -m pip install --upgrade pip
    python -m pip install pytest playwright pytest-playwright
    python -m pip install pytest pytest-playwright

5. Make sure directory to "pages" folder
6. Run the file "test_sit_pc.py" using command ---- python -m pytest test_sit_pc.py
